# 试玩反馈与二轮迭代路线

> 来源：第一次本地试玩后的集中反馈。目标是把当前“能玩”的原型推进到“看得清、打得爽、有成长、有运气、有流派差异”的阶段。

## 一、当前核心问题

### 1. 经验光球与怪物辨识度冲突

当前经验掉落物和怪物大小接近，且视觉表现偏花，导致玩家在战斗中分不清“经验”与“敌人”。这会直接破坏操作判断和战场阅读。

调整方向：

- 经验掉落物改为更小、更简单的蓝色菱形晶体。
- 不再做复杂装饰、发光和角色化表现。
- 经验晶体只承担“拾取资源”的视觉功能，不能和怪物抢视觉层级。
- 生命掉落可以继续保留绿色，但同样要保持简洁。

优先级：最高。

### 2. 角色与种族美术辨识度不足

当前人族、精灵、哥布林、兽人、灵族等种族虽然有资源，但局内主体仍然太抽象。玩家无法一眼看出：

- 人族应该像人。
- 精灵应该有尖耳、轻盈、灵巧感。
- 兽人应该更壮、更厚重。
- 哥布林应该更小、更狡猾。
- 灵族应该有半透明、飘浮、灵体感。

调整方向：

- 种族 sprite 要强化轮廓差异，而不是只换颜色。
- 局内角色应该在小尺寸下也能读出身份。
- 角色大小、比例、头身、耳朵、肩宽、腿部、武器握持姿态都要体现差异。

优先级：高。

### 3. 怪物强度不足，AI 智商偏低

当前怪物整体威胁不够，尤其玩了几波后，敌人只是直线靠近，套路单一。精英和 Boss 也没有真正形成压迫感。

主要问题：

- 普通怪只会追人，缺少包围、侧翼、压迫路线。
- 精英怪和 Boss 子弹太少，弹幕压力不足。
- 不同怪物之间差异不明显。
- 波次成长不够明显，后期压力提升不够。

调整方向：

- 加入围杀 AI：部分敌人不直接冲脸，而是绕到玩家侧面或背后。
- 加入分工：冲锋怪、射手怪、坦克怪、包围怪、召唤怪、爆炸怪。
- 精英怪应拥有更高攻击频率、更强技能或特殊移动方式。
- Boss 需要多阶段或多模式攻击，例如三连发、扇形弹幕、环形弹幕、召唤小怪。
- 怪物强度随波次提升更明显，不只是血量提高。

优先级：最高。

### 4. 升级强化缺少稀有度与运气波动

当前升级选择已经有通用强化和体系/武器特定强化，这是对的，但还缺少“这把运气真好”的感觉。

需要加入强化稀有度：

- 普通级：基础数值提升。
- 稀有级：明显优于普通，有额外效果。
- 史诗级：改变手感或机制。
- 传说级：强力流派核心。
- 钻石级：极低概率出现，效果非常夸张，是爽感爆点。

例子：

- 无人机编队普通级：增加 1 架无人机。
- 无人机编队黄金级：增加 2 架无人机，攻击频率提高。
- 无人机编队钻石级：无人机数量大幅增加，并附带穿透或爆炸。

还需要加入：

- 升级界面刷新按钮。
- 刷新需要消耗某种资源或限制次数。
- 稀有强化出现时要有特殊边框、颜色、音效和提示。

优先级：高。

### 5. 爽感来源需要围绕“成长爆炸”设计

游戏后期的核心爽感应该来自：玩家运气好、成长方向正确、体系成型后，能大范围清怪。

示例方向：

- 魔杖/法杖流：后期一次攻击打倒一大片怪，视觉应该绚丽，有连锁闪电、爆炸、奥术弹幕、魔法阵。
- 弓箭流：多重箭、追踪箭、爆炸箭、穿透箭形成弹幕压制。
- 飞刃/剑技流：高阶后出现剑气、回旋斩、剑阵、斩击波。
- 钝器/狼牙棒流：攻击慢但范围大，破甲强，能击退或震地。
- 长枪流：穿刺、突进、直线贯通、枪芒。
- 科技/无人机流：无人机数量、锁定、激光、导弹、自动清场。

核心目标：

- 不是只加数值，而是每个体系高阶后要有不同视觉和手感。
- 后期应该明显感觉“我这把成型了”。
- 强化之间要能组合，形成流派闭环。

优先级：最高。

## 二、二轮迭代优先级

### P0：立刻修正

1. 经验晶体小型化、简洁化。
2. 怪物/掉落物视觉层级分离。
3. 初步提高怪物强度和波次压力。
4. 精英/Boss 发射更多子弹，形成压迫感。

### P1：核心玩法增强

1. 怪物围杀 AI。
2. 怪物分工与行为差异。
3. 升级强化稀有度系统。
4. 升级刷新按钮。
5. 属性面板显示更完整。

### P2：爽感与流派成型

1. 魔法流大范围绚丽特效。
2. 剑技/飞刃高阶剑气。
3. 钝器破甲、击退、震地。
4. 科技/无人机后期自动清场。
5. 钻石级强化的特殊表现。

## 三、设计原则

### 1. 视觉层级必须清楚

玩家、敌人、掉落物、弹体、特效不能混在一起。重要程度排序：

1. 玩家。
2. 敌人和危险弹体。
3. 玩家弹体。
4. 掉落物。
5. 装饰特效。

### 2. 怪物不能只是移动血包

怪物要制造决策压力：

- 我该往哪跑？
- 我先杀远程还是近战？
- 我能不能贪经验？
- Boss 弹幕来了我要不要撤？

### 3. 强化不是只有数值，而是改变玩法

好的强化应该让玩家感觉操作方式变了，例如：

- 弓箭变成扇形弹幕。
- 魔杖变成连锁爆炸。
- 剑技变成屏幕斩击。
- 无人机变成自动军团。

### 4. 运气要制造惊喜，但不能完全失控

稀有强化要有随机性，但也要能服务流派：

- 当前武器相关强化权重更高。
- 通用强化兜底。
- 稀有强化低概率出现。
- 刷新按钮给玩家一点主动权。

## 四、下一步开发切入点

推荐立刻开发顺序：

1. 改经验晶体资源和拾取渲染尺寸。
2. 调整怪物基础强度和 Boss 弹幕。
3. 给敌人加入简单围杀行为。
4. 加强化稀有度数据结构。
5. 升级面板加入刷新按钮。
6. 给不同稀有度加 UI 边框和颜色。
7. 开始补每个体系的高阶特效。
