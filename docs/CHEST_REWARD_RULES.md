# 宝箱材料奖励规范

本文件用于防止后续开发重新混淆“局内补给”和“宝箱奖励”。

优先级参考：`docs/PROGRESSION_CORRECTIONS.md`。

## 一、硬规则

宝箱不能掉局内临时补给。

禁止放入宝箱奖励池的内容包括：

```text
磁铁
聚宝符
血包
护盾
回春露
净化符
急速药剂
攻击药剂
暴击药剂
狂暴药剂
冰霜炸弹
雷击符石
地裂符石
诱饵人偶
佣兵号角
亡骨号角
机械炮台
临时召唤物
临时炮台
```

这些属于 `src/data/runItems.ts` 管理的局内补给，只能通过地图刷新、普通怪低概率掉落、事件、商人、祭坛或随机房间进入单局。

## 二、宝箱应该掉什么

宝箱服务长期成长，掉落内容应该可以带出局并进入材料仓库。

允许内容：

```text
普通材料
稀有材料
珍贵材料
神话生物骨骼
异种核心
蓝图
残页
符文
职业解锁物
天赋解锁材料
区域收复材料
合成配方相关材料
```

## 三、宝箱类型建议

### 小宝箱

来源：精英怪概率掉落、特殊事件、区域任务。

建议内容：

```text
普通材料 x2-4
稀有材料 x0-1
区域材料 x0-1
残页碎片 x0-1
```

### 大宝箱

来源：Boss 必掉、区域首通奖励、高风险挑战。

建议内容：

```text
普通材料 x4-8
稀有材料 x1-3
珍贵材料 x0-1
蓝图 / 残页 / 职业解锁物 x0-1
区域收复材料 x1
```

### 神话宝箱

来源：区域 Boss、隐藏事件、噩梦以上难度、成就任务。

建议内容：

```text
珍贵材料 x1-2
神话生物骨骼 x0-1
异种核心 x0-1
职业解锁物 x1
高阶蓝图 x0-1
```

## 四、UI 命名规范

为避免玩家误解，UI 中也要区分：

```text
局内补给：临时效果 / 本局有效 / 补给
宝箱奖励：可带出材料 / 长期成长 / 仓库材料
```

不要在宝箱开箱界面出现：

```text
获得血包
获得急速药剂
获得临时炮台
```

应该出现：

```text
获得异种残核 x3
获得王骸晶核 x1
获得炮塔蓝图 x1
获得元素残页 x2
```

## 五、和长期成长系统的关系

宝箱材料主要流向：

1. 天赋解锁。
2. 天赋槽购买。
3. 职业路线解锁。
4. 道具工坊升级。
5. 药房研究。
6. 合成系统。
7. 土地收复地图。

宝箱不是单局补给站，而是长期成长入口。
