# 局外成长系统文件索引

本文件用于说明当前局外成长、经济、天赋、合成相关文件的职责。

## 设计规则文档

### docs/PROGRESSION_CORRECTIONS.md

长期成长系统纠正规则。

重点：

- 局内补给和宝箱奖励必须分开。
- 天赋槽不是按波次白送。
- 新手引导免费送第一个天赋槽。
- 后续天赋槽位需要购买。
- 主界面应该是远征营地。

### docs/ECONOMY_RULES.md

经济分层规则。

重点：

- 玩家带出局资源分为通用物品和特殊物品。
- 通用物品用于购买天赋、普通槽位、基础工坊、药房和营地升级。
- 特殊物品用于神话武器、永久药剂、职业核心、高阶天赋、高级槽位和区域关键节点。

### docs/PROGRESSION_SYSTEMS.md

早期长期成长规划文档。

注意：如果和 `PROGRESSION_CORRECTIONS.md` 或 `ECONOMY_RULES.md` 冲突，以后两者为准。

## 数据表文件

### src/data/runItems.ts

局内补给道具数据表。

包括：

- 磁铁
- 血包
- 护盾
- 急速药剂
- 攻击药剂
- 暴击药剂
- 冰霜炸弹
- 雷击符石
- 诱饵人偶
- 机械炮台

注意：这些是局内临时补给，不是宝箱主要奖励。

### src/data/hubModules.ts

远征营地主界面模块数据表。

包括：

- 远征入口
- 天赋殿堂
- 道具工坊
- 药房
- 任务系统
- 合成系统
- 材料仓库
- 收复地图
- 异种档案

### src/data/economy.ts

局外经济资源数据表。

分为：

- general：通用物品 / 通用货币
- special：特殊物品 / 稀有材料

### src/data/metaTalents.ts

局外四字天赋数据表。

规则：

- 所有天赋名称必须四个字。
- 普通天赋主要消耗通用资源。
- 职业天赋和高阶天赋才消耗特殊物品。

### src/data/craftingRecipes.ts

合成系统配方表。

合成目标包括：

- 神话武器胚子
- 永久药剂
- 职业核心
- 天赋槽扩展券
- 工坊项目
- 药房项目
- 区域关键物
