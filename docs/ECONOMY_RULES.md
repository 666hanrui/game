# 经济系统规则：通用物品与特殊物品

本文件用于明确玩家带出局资源的分层规则。后续天赋、合成、工坊、药房、神话武器、永久药剂都要遵守这里的经济分层。

## 核心原则

玩家带出局的东西不能全部混成“材料”。

正确分为两类：

1. 通用物品 / 通用货币
2. 特殊物品 / 稀有材料

通用物品负责基础成长，让玩家每一局都有稳定收益。

特殊物品负责高阶追求，让神话武器、永久药剂、职业核心、区域关键节点这些东西有长期目标感。

## 通用物品 / 通用货币

通用物品是玩家每局都能比较稳定获得的资源。

来源：

- 普通清剿奖励
- 波次结算
- 普通任务
- 小宝箱少量产出
- 区域收复基础奖励
- 分解低级材料

用途：

- 购买天赋
- 升级普通天赋
- 购买前期天赋槽
- 普通工坊升级
- 普通药房研究
- 基础营地升级
- 刷新局外商店

设计目的：

通用物品保证玩家每次远征结束都能做一点事，不会因为没掉珍贵材料就完全卡住。

### 示例通用物品

#### 远征币

最基础的局外货币。

用途：购买普通天赋、普通天赋升级、基础设施升级。

#### 魂晶

从异种身上回收的能量结晶。

用途：天赋升级、药房研究、工坊研究。

#### 异种残核

普通异种留下的残余核心。

用途：低级天赋解锁、基础工坊项目。

#### 营地物资

远征营地建设用的普通物资。

用途：主界面设施升级、基础功能开放。

## 特殊物品 / 稀有材料

特殊物品是高价值资源，不应该被普通天赋购买大量消耗掉。

来源：

- Boss 大宝箱
- 精英小宝箱
- 封印宝箱
- 高难度任务
- 区域事件
- 隐藏挑战
- 特殊区域 Boss

用途：

- 神话武器合成
- 永久性提升药
- 职业核心解锁
- 高阶天赋突破
- 高级天赋槽扩展
- 特殊种族进阶
- 高级工坊项目
- 高级药房研究
- 区域收复关键节点

设计目的：

特殊物品制造长期目标和稀有感。玩家看到这些物品时，应该想到“这东西能合成大东西”，而不是拿去买普通天赋。

### 示例特殊物品

#### 神话生物骨骼

用途：神话武器、死灵路线、古武传承、永久提升药、高阶合成。

#### 王骸晶核

用途：Boss 相关高阶天赋、职业核心、区域关键节点、高级天赋槽。

#### 星陨晶髓

用途：科技路线、能量核心、法球路线、高级药剂。

#### 古代兵装残片

用途：长枪、狼牙棒、飞刃等古武高阶路线。

#### 亡者冠片

用途：死灵法师路线、亡灵相关天赋。

#### 炮塔蓝图

用途：机械组、炮塔系统、工坊高级项目。

#### 元素残页

用途：元素术士、元素组合、魔法路线。

#### 猎王徽记

用途：游侠高阶路线、猎杀类天赋、精英挑战。

## 系统消耗规则

### 天赋购买

普通天赋：主要消耗通用货币。

高级天赋：通用货币 + 少量对应特殊物品。

职业天赋：通用货币 + 职业解锁物。

### 天赋槽位

第 1 个天赋槽：新手引导免费赠送。

前期槽位：主要消耗通用货币。

高级槽位：通用货币 + 稀有特殊物品。

不要让所有槽位都直接吃神话级材料，否则玩家会觉得经济系统卡死。

### 神话武器

神话武器合成必须以特殊物品为主。

可以少量消耗通用货币作为手续费，但核心材料必须是稀有物。

### 永久药剂

永久性提升药是高阶成长，不能用普通货币直接买。

应该消耗：特殊物品 + 药房研究等级 + 合成配方。

### 道具工坊

普通补给强化：通用货币为主。

高级补给解锁：通用货币 + 对应特殊物品。

例如临时炮台、亡骨号角、诱饵人偶这类东西，可以吃蓝图或特殊物品。

### 药房

普通药剂研究：通用货币为主。

永久药剂与高级药剂：特殊物品为主。

### 合成系统

合成系统是特殊物品的主要出口。

合成结果必须服务明确目标：

- 神话武器
- 永久药剂
- 职业核心
- 高阶天赋
- 高级天赋槽
- 区域收复
- 高级工坊项目

## 反例

以下设计不要采用：

- 用神话生物骨骼购买普通天赋。
- 宝箱里掉血包、磁铁、攻速药剂这类局内补给。
- 所有系统都消耗同一种珍贵材料。
- 玩家连续几局没有特殊材料就完全无法成长。
- 通用货币没有用途，只能堆在仓库里。

## 正确体验目标

每局结束后，玩家应该同时获得两种感觉：

1. 通用资源变多了，可以马上买点小东西。
2. 特殊物品慢慢攒起来，未来能合成真正厉害的东西。

这两条线必须并行存在。通用资源负责短期反馈，特殊物品负责长期追求。
