# 远征营地主界面 UI 设计草图

本文件以 `docs/PROGRESSION_CORRECTIONS.md` 为准，主界面不再设计成普通菜单，而是玩家每次退出单局后返回的“远征营地”。

## 一、核心定位

远征营地是长期成长系统的总入口，承担三个职责：

1. 开始下一局远征。
2. 消化上一局带出的材料。
3. 展示长期目标，例如天赋、职业、区域收复、图鉴和任务。

注意：

- 局内补给和宝箱奖励必须分开。
- 局内补给只影响单局战斗节奏。
- 宝箱只给长期成长材料。
- 天赋槽不是打到 6 波自动送。
- 新手引导免费给第 1 个天赋槽，后续槽位用珍贵材料购买。

## 二、主界面整体布局

推荐分成 5 个区域：

```text
┌────────────────────────────────────────────────────────────┐

├──────────────┬──────────────────────────┬──────────────────┤
│ 左侧导航模块 │ 中央远征营地视觉区         │ 右侧情报面板       │
│              │                          │                  │
│ 远征入口     │        远征营地场景        │ 当前区域           │

│ 道具工坊     │                          │ 可能特殊怪         │
│ 药房         │                          │ 可能材料掉落       │
```

## 三、模块视觉设计

### 1. 远征入口

定位：最重要主按钮，视觉优先级最高。

建议表现：

- 放在中央偏下或右下的大按钮。
- 使用金色 / 火光 / 旗帜图标。
- 文案：`开始远征`。
- 附带当前区域、推荐难度、特殊怪、可掉落材料、当前可携带天赋槽数量。

示例信息：

```text
当前区域：破碎平原
推荐难度：正常
特殊怪：爆炸怪 / 治疗怪
可能材料：异种残核、腐化骨片、裂土印记
天赋槽：1 / 3
```

### 2. 天赋殿堂

定位：长期成长核心。

建议表现：

- 使用石柱、星盘、符文门的视觉。
- 明确显示“第 1 槽位：新手引导赠送”。
- 后续槽位显示材料购买条件，避免玩家误以为打波次自动获得。

槽位展示建议：

```text
槽位 1：已解锁（新手引导）
槽位 2：需要 王骸晶核 x1 + 星陨晶髓 x1
槽位 3：需要 神话生物骨骼 x2 + 裂土印记 x4
```

### 3. 道具工坊

定位：强化局内补给池。

注意：工坊强化不等于宝箱会掉补给。

建议表现：

- 图标使用铁砧、齿轮、工具。
- 子项包括磁铁范围、护盾强度、诱饵人偶、临时炮台。
- 文案要写清楚：`影响局内补给刷新池`。

### 4. 药房

定位：药剂类局内补给研究。

建议表现：

- 绿色 / 蓝色瓶罐、药柜。
- 子项包括血包、回春露、急速药剂、攻击药剂、暴击药剂、净化符。
- 文案：`研究后进入局内补给池`。

### 5. 任务系统

定位：给玩家阶段目标。

建议表现：

- 任务板、羊皮纸、印章。
- 分类：清剿、精英、Boss、材料、体系试炼、高风险挑战、区域收复。
- 奖励显示材料、残页、蓝图、区域进度、天赋槽折扣。

### 6. 合成系统

定位：把带出局材料转化成长期成长钥匙。

建议表现：

- 炼成台、配方盘、材料槽。
- 输入材料在左，产物在右。
- 产物可以是职业核心、天赋解锁物、区域钥印、工坊升级物。

### 7. 材料仓库

定位：查看所有带出局材料。

建议表现：

- 仓库格子、材料筛选、稀有度颜色。
- 每个材料要显示来源、用途、数量、可合成目标。
- 不显示局内临时补给，避免系统混淆。

### 8. 土地收复地图

定位：长期地图进度。

建议表现：

- 中央大地图或右侧大地图入口。
- 区域节点显示收复进度、推荐难度、特殊怪组合、可掉落材料、区域 Boss。

区域示例：

```text
破碎平原：收复 35%
腐化林地：收复 0%
骸骨荒原：未解锁
晶化废土：未解锁
王巢核心：未解锁
```

### 9. 图鉴与异种档案

定位：收集和世界观补充。

建议表现：

- 档案柜、怪物剪影、材料图鉴。
- 记录怪物、Boss、材料、职业路线和区域故事。

## 四、远征营地视觉风格

推荐关键词：

- 暗色背景。
- 篝火暖光。
- 蓝色材料晶体冷光。
- 营地建筑带轻微图标化。
- 模块按钮像营地建筑牌子，而不是普通 UI 菜单按钮。

颜色建议：

```text
背景：#0b1020 / #101827
主按钮：#ffca28 / #ff8f00
材料：#90caf9 / #b388ff / #ffd54f
长期成长：#ce93d8
局内补给：#66bb6a / #42a5f5
危险区域：#ef5350
```

## 五、信息层级

首屏优先级：

1. 开始远征。
2. 当前区域和推荐难度。
3. 本区域可能掉落的长期材料。
4. 当前天赋槽数量。
5. 任务提醒。
6. 其他成长模块入口。

不要把所有系统平铺成同等权重，否则新玩家会被长期成长系统吓住。

## 六、宝箱和补给的 UI 区分

必须在 UI 语言上强制区分：

### 局内补给

用于单局即时效果。

视觉：

- 小图标。
- 带持续时间。
- 常用绿色、蓝色、黄色。
- 文字使用“局内补给”“临时效果”。

示例：

```text
血包、护盾、磁铁、急速药剂、诱饵人偶、临时炮台
```

### 宝箱材料

用于长期成长。

视觉：

- 宝箱、材料晶体、骨骼、蓝图、残页。
- 带稀有度边框。
- 进入材料仓库。
- 文字使用“可带出材料”“长期成长材料”。

示例：

```text
异种残核、腐化骨片、王骸晶核、神话生物骨骼、星陨晶髓、炮塔蓝图、元素残页
```

## 七、首版实现建议

第一版不需要做复杂建筑动画，可以先做静态 Canvas UI：

1. 顶部资源栏。
2. 左侧 9 个模块按钮。
3. 中央营地插画占位。
4. 右侧远征情报卡。
5. 底部开始远征按钮。

后续再把模块按钮替换成可点击建筑，逐步演化成真正的远征营地。
