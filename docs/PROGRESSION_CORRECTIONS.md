# 长期成长系统纠正规则

本文件优先级高于 `docs/PROGRESSION_SYSTEMS.md` 中的旧规则。

## 一、局内补给和宝箱奖励必须分开

局内补给用于改变单局战斗节奏。

包括但不限于：

- 磁铁
- 血包
- 护盾
- 急速药剂
- 攻击药剂
- 暴击药剂
- 净化符
- 冰霜炸弹
- 雷击符石
- 诱饵人偶
- 临时炮台
- 临时召唤物

这些东西可以来自地图刷新、普通怪低概率掉落、特殊事件、商人、祭坛或随机房间。

宝箱奖励用于长期成长，不能掉这些普通补给。

宝箱应该产出：

- 普通材料
- 稀有材料
- 珍贵材料
- 神话生物骨骼
- 异种核心
- 蓝图
- 残页
- 符文
- 职业解锁物
- 天赋解锁材料
- 区域收复材料

一句话：补给服务局内爽感，宝箱服务长期成长。

## 二、天赋槽位规则修正

天赋槽不是按波次自动赠送。

正确规则：

- 新手引导阶段，玩家升级到指定等级后，免费获得第 1 个天赋槽。
- 第 1 个槽位用于教学，让玩家明白天赋可以带入下一局。
- 后续天赋槽位都要购买。
- 购买天赋槽位消耗局内带出的珍贵材料。
- 天赋本身也需要通过材料解锁、任务奖励、合成系统或职业路线获得。
- 开局只能装备当前已经拥有的天赋槽数量。
- 所有天赋名称必须是四个字。

## 三、主界面应该是远征营地

玩家退出单局后，回到的不是普通菜单，而是远征营地。

远征营地至少包含以下模块。

### 远征入口

用于开始新一局游戏。

显示：

- 当前区域
- 推荐难度
- 可能出现的特殊怪
- 可能掉落的材料
- 当前可携带天赋槽数量

### 天赋殿堂

用于查看、解锁、购买、装备天赋。

核心规则：

- 新手引导送第 1 个天赋槽
- 后续槽位用材料购买
- 天赋用材料、任务、合成或职业路线解锁
- 强力天赋可以带负面效果

### 道具工坊

用于强化局内补给池。

例如：

- 提高磁铁范围
- 提高护盾强度
- 延长急速药剂时间
- 解锁诱饵人偶
- 解锁临时炮台

注意：工坊强化的是局内补给系统，不代表宝箱会掉补给。

### 药房

用于研究药剂类局内补给。

例如：

- 血包
- 回春露
- 急速药剂
- 攻击药剂
- 暴击药剂
- 狂暴药剂
- 净化符

药房可以影响药剂是否进入局内刷新池，或提高某类药剂出现权重。

### 合成系统

用于把材料合成为更高级的材料、职业核心、天赋解锁物、区域钥印或工坊升级物。

合成系统对应宝箱中产出的神话生物骨骼、异种核心、蓝图、残页、符文等材料。

示例：

- 腐化骨片 + 神话生物骨骼 + 亡者冠片 = 亡骸印记
- 机械残芯 + 炮塔蓝图 + 星陨晶髓 = 炮台核心
- 古代兵装残片 + 神话生物骨骼 + 王骸晶核 = 古武传承
- 元素残页 + 月银枝骨 + 星陨晶髓 = 元素秘卷
- 猎王徽记 + 月银枝骨 + 裂土印记 = 猎王誓印
- 裂土印记 + 王骸晶核 = 天赋槽扩展材料

### 任务系统

用于给玩家阶段目标。

任务类型：

- 清剿任务
- 精英任务
- Boss 任务
- 材料任务
- 体系试炼
- 高风险挑战
- 区域收复任务

奖励可以是材料、天赋残页、职业蓝图、区域收复进度、天赋槽购买折扣。

### 材料仓库

用于查看所有可带出局材料。

显示：

- 名称
- 稀有度
- 来源
- 用途
- 当前数量
- 可合成目标

### 土地收复地图

用于展示世界地图和区域进度。

区域示例：

- 破碎平原
- 腐化林地
- 骸骨荒原
- 晶化废土
- 王巢核心

每个区域有：推荐难度、特殊怪组合、可掉落材料、区域 Boss、收复进度、区域事件。

### 图鉴与异种档案

用于记录玩家见过的怪物、Boss、材料、职业路线和区域故事。

1. 天赋解锁

4. 区域收复与合成

材料示例：

### 珍贵材料

### 职业解锁物

- 猎王徽记

## 五、天赋设计方向

所有天赋名称必须是四个字。

示例：

### 连斩追击

连续击杀时叠加伤害或攻速。

### 血影回流

击杀或暴击时回复少量生命。

### 孤注一击

攻击力大幅提升，但最大生命固定为 1，受击即失败。

### 亡骸归阵

击杀敌人有概率生成短时亡灵随从。

### 炮台矩阵

每隔一段时间部署临时炮塔。

### 贪宝成瘾

宝箱材料奖励提高，但怪物强度提高。

### 逆命狂热

生命越低，攻击和移速越高，但治疗效果降低。

### 晶盾回响

护盾破裂时对周围敌人造成冲击，但最大生命降低。

## 六、职业方向

### 死灵法师

核心爽点：一点点积累自己的亡灵大军。

机制方向：击杀召唤骷髅、精英转化强随从、Boss 掉落死灵职业材料。

### 机械组

核心爽点：炮塔、防线、无人机、自动火控。

机制方向：临时炮台、炮塔升级、炮塔电网、机械核心合成。

### 狂战士

核心爽点：低生命高爆发。

机制方向：生命越低越强、击杀续航、风险天赋联动。

### 元素术士

核心爽点：冰、火、雷、奥术组合。

机制方向：冰冻、爆裂、连锁、奥术新星、元素二段反应。

### 游侠

核心爽点：箭雨、追踪、陷阱、点杀支援怪。

机制方向：多重箭、标记、陷阱、精英猎杀。
