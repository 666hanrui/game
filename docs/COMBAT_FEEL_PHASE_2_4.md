# 战斗爽感第二三四阶段执行说明

本文件记录 `docs/COMBAT_FEEL_REWORK_PLAN.md` 的第二、第三、第四阶段落地内容。

## 第二阶段：升级卡派生攻击

玩家层仍然只看到“升级卡”，没有新增形态卡类型。

### 已调整升级卡

```text
src/data/skills.ts
```

新增或调整：

```text
枪芒：长枪刺击后额外释放远程枪芒。
地裂：狼牙棒重击后向前释放地裂波。
震荡波：狼牙棒重击时释放环形震荡。
```

这些都是普通升级卡，不是新的卡种。

### 已接入运行时

```text
src/core/Game.ts
```

当前接入：

```text
spear + spear_shadow -> releaseSpearBeam()
mace + earthquake -> releaseMaceQuake()
mace + mace_shockwave -> releaseMaceShockwaves()
```

长枪和狼牙棒的基础攻击仍然是近战。

枪芒、地裂、震荡波是在近战攻击后额外产生的派生攻击。

## 第三阶段：天赋元素标签

元素不进入局内升级卡池。

元素倾向现在放在局外天赋：

```text
src/data/metaTalents.ts
```

新增元素 / 倾向天赋：

```text
雷纹共鸣 -> lightning
炎脉涌动 -> fire
霜息凝痕 -> ice
毒蚀蔓延 -> poison
奥术星环 -> arcane
机巧制导 -> tech
```

这些英文标签是代码内部联动条件，玩家仍然只看到天赋说明。

## 第四阶段：隐藏联动第一版

新增：

```text
src/data/synergies.ts
src/systems/SynergyRuntime.ts
```

联动不是卡牌，不会进入升级池。

代码会检测：

```text
已拥有升级卡 + 已装备天赋标签
```

然后自动激活对应联动。

### 第一批联动

```text
多重箭 + 雷纹共鸣 = 雷羽分裂
冰霜箭 + 霜息凝痕 = 霜痕箭路
追踪箭 + 机巧制导 = 制导箭
爆裂箭 + 炎脉涌动 = 炎爆箭
枪芒 + 雷纹共鸣 = 雷贯枪芒
枪芒 + 霜息凝痕 = 寒星枪芒
地裂 + 炎脉涌动 = 熔岩地裂
地裂 + 霜息凝痕 = 冰裂大地
回旋飞刃 + 毒蚀蔓延 = 毒刃回旋
无人机 + 奥术星环 = 奥术无人机
```

## 战斗内实际效果

已接入：

```text
src/core/Game.ts
```

当前表现：

```text
雷羽分裂：箭矢命中后向附近敌人跳电。
霜痕箭路：冰霜箭减速更明显，并有冰蓝碎片。
制导箭：追踪箭锁定范围和转向强度提高。
炎爆箭：爆裂箭追加火焰粒子和小范围火爆。
雷贯枪芒：枪芒命中后跳电。
寒星枪芒：枪芒命中后减速和冰霜粒子。
熔岩地裂：地裂波命中后追加火焰爆裂。
冰裂大地：地裂波命中后追加减速和碎冰。
毒刃回旋：飞刃命中后减速并有毒蚀粒子。
奥术无人机：无人机弹体转为奥术弹，并增加命中反馈。
```

## 本地重点验证

```text
1. 长枪拿到“枪芒”后，近战刺击仍在，同时会额外发出枪芒。
2. 狼牙棒拿到“地裂”后，近战重击仍在，同时会向前释放地裂波。
3. 狼牙棒拿到“震荡波”后，会出现环形冲击。
4. 装备雷纹共鸣后，多重箭 / 枪芒是否出现跳电。
5. 装备炎脉涌动后，爆裂箭 / 地裂是否出现火焰反馈。
6. 装备霜息凝痕后，冰霜箭 / 枪芒 / 地裂是否减速更明显。
7. 装备毒蚀蔓延后，回旋飞刃是否有绿色毒蚀反馈。
8. 装备机巧制导后，追踪箭是否更明显追踪。
9. 升级面板是否出现“选择后联动 / 已激活联动”提示。
```

## 设计边界

```text
不要把元素做成局内升级卡。
不要新增形态卡、机制卡、联动卡。
枪芒、地裂、震荡波都只是普通升级卡效果。
隐藏联动由代码自动触发。
```
