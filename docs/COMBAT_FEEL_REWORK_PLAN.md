# 战斗爽感与构筑系统重构计划

## 一、总目标

当前战斗存在几个核心问题：

1. 很多武器攻击表现都像圆形小球，辨识度不足。
2. 近战武器前期不应该像远程武器一样发射弹体。
3. 升级卡重复获得后提升不明显，缺少“再次抽到也很爽”的感觉。
4. 流派成型感不足，玩家不容易感受到“我的体系已经搭起来了”。
5. 不同升级、武器、天赋之间缺少隐藏联动，构筑探索感不足。
6. 命中反馈偏弱，攻击打到怪物身上不够爽。

重构目标：

```text
武器决定基础攻击方式；
升级卡负责武器机制成长；
天赋负责元素倾向和特殊底色；
代码自动检测隐藏联动；
流派阶段负责初成 / 成型 / 大成反馈；
玩家只看到简单的武器、升级卡、天赋，但战斗中自然打出复杂又爽的效果。
```

最终体验应该是：

```text
弓箭射出去就是箭；
魔法打出去像魔法；
科技攻击有科技感；
长枪前期就是近战刺击；
狼牙棒前期就是近战重击；
剑前期就是近战斩击；
拿到对应升级后，近战武器才获得枪芒、剑气、地裂等派生攻击；
天赋让这些升级自动产生隐藏联动；
中后期玩家能明显感受到流派成型和杂交大成。
```

## 二、玩家层系统只保留三类

为了避免系统越来越乱，玩家可见层只保留三类：

```text
1. 武器
2. 升级卡
3. 天赋
```

不要额外新增：

```text
形态解锁卡
元素卡
机制改造卡
联动触发卡
```

这些概念可以存在于代码内部，但不要作为新的卡牌类型暴露给玩家。

玩家理解方式应该是：

```text
我选了一把武器。
我在局内拿升级卡强化它。
我在局外或局内通过天赋确定元素倾向。
当升级卡和天赋匹配时，游戏自动打出隐藏联动。
```

## 三、代码层隐藏结构

代码里可以复杂，但玩家不需要直接理解。

建议代码层包含：

```text
WeaponAttackMode        武器基础攻击形态
ProjectileVisualType    远程弹体视觉类型
MeleeAttackType         近战攻击类型
UpgradeTags             升级卡标签
TalentTags              天赋标签
SynergyRegistry         自动联动表
BuildStageRuntime       流派阶段统计
HitFeedbackRuntime      命中特效反馈
```

大原则：

```text
复杂度藏在代码里；
爽感显示在战斗里；
玩家 UI 保持简单。
```

## 四、武器基础攻击形态

### 1. 弓箭

弓箭天生远程。

初始攻击：

```text
发射细长箭矢；
箭矢朝飞行方向旋转；
带短尾羽；
飞行时有轻微拖尾；
不再是圆球。
```

可吃升级：

```text
分裂箭
穿透箭
追踪箭
多重箭
爆裂箭
箭雨
速射
精准射击
```

### 2. 魔杖 / 法杖 / 法球

魔法类天生远程或自动化，但要区分视觉。

魔杖：

```text
小型紫蓝奥术弹；
速度快；
带粒子尾巴；
偏连续输出。
```

法杖：

```text
较大的符文飞弹；
更亮更重；
命中时出现小法阵或符文爆开；
偏范围和爆发。
```

法球：

```text
发光能量球；
可环绕、波纹、护盾感；
偏自动化和防守反击。
```

### 3. 科技 / 能量核心 / 无人机核心

科技类要有机械感，不能还是小圆球。

能量核心：

```text
菱形或六边形能量弹；
蓝白色；
带扫描线；
命中时有电子爆裂。
```

无人机核心：

```text
小三角脉冲弹；
无人机自动射击；
可以有锁定框和齐射效果。
```

### 4. 长枪

长枪前期必须是近战。

初始攻击：

```text
近距离刺击；
没有远程弹体；
攻击范围是一条短直线；
视觉是枪尖刺击光 / 枪刺线。
```

局内升级卡“枪芒”：

```text
长枪刺击后，额外释放一道向前飞行的枪芒。
```

注意：

```text
枪芒不是新卡牌类型；
它只是普通升级卡。
```

### 5. 剑 / 刃

普通剑前期必须是近战。

初始攻击：

```text
近距离扇形斩击；
没有远程剑气；
视觉是扇形斩击弧光。
```

局内升级卡“剑气”：

```text
斩击后额外飞出一道剑气。
```

飞刃可以特殊一些：

```text
飞刃初始可以是短距离投刃 / 回旋刃；
它不是普通远程弹体，而是短距离回旋或返回机制。
```

### 6. 狼牙棒 / 钝器

狼牙棒前期必须是重近战。

初始攻击：

```text
近距离重击；
小范围圆形或扇形打击；
高击退；
低攻速；
没有远程弹体。
```

局内升级卡“地裂”：

```text
重击后向前释放一道地裂波。
```

局内升级卡“震荡波”：

```text
重击或命中后释放环形震荡。
```

同样注意：

```text
地裂、震荡波都只是普通升级卡，不是新卡种。
```

## 五、升级卡设计原则

升级卡只负责“武器机制成长”。

不再把元素放进升级卡池。

升级卡可以做：

```text
改变攻击方式；
增加派生攻击；
增加数量；
增加穿透；
增加追踪；
增加范围；
增加爆裂；
增加回旋；
增加枪芒 / 剑气 / 地裂等武器派生。
```

升级卡不能变成：

```text
雷元素卡
火元素卡
冰元素卡
毒元素卡
```

这些交给天赋。

## 六、升级卡三段式

同一张升级卡重复获得时，不应该只是小幅数值增加，而应该进入阶段成长。

统一规则：

```text
I 级：解锁机制
II 级：明显强化
III 级：质变效果
```

### 示例：分裂箭

```text
分裂箭 I：
箭矢命中后分裂 2 支小箭。

分裂箭 II：
分裂 3 支小箭，小箭伤害提高。

分裂箭 III：
小箭继承更多天赋联动效果，分裂角度更稳定。
```

### 示例：追踪箭

```text
追踪箭 I：
箭矢获得基础追踪，小角度修正目标。

追踪箭 II：
追踪更灵敏，锁定范围扩大。

追踪箭 III：
命中后可二次索敌，或分裂出小追踪箭。
```

### 示例：穿透箭

```text
穿透箭 I：
箭矢穿透 +1。

穿透箭 II：
穿透后伤害衰减降低。

穿透箭 III：
穿透后裂出碎箭，或路径效果继承天赋联动。
```

### 示例：枪芒

```text
枪芒 I：
长枪刺击后发出一道枪芒。

枪芒 II：
枪芒更长，可穿透 1 个敌人。

枪芒 III：
枪芒继承更多天赋联动，命中后触发额外效果。
```

### 示例：剑气

```text
剑气 I：
斩击后飞出一道剑气。

剑气 II：
剑气更宽，飞行更远。

剑气 III：
剑气可分裂或返回，并继承天赋联动。
```

### 示例：地裂

```text
地裂 I：
狼牙棒重击后释放一道地裂波。

地裂 II：
地裂范围更宽，击退更强。

地裂 III：
地裂留下短暂地面效果，并继承天赋联动。
```

## 七、天赋负责元素倾向

元素不要进入升级卡池，改成天赋系统的一部分。

天赋提供元素或特殊倾向标签：

```text
雷纹天赋：lightning
炎脉天赋：fire
霜息天赋：ice
毒蚀天赋：poison
奥术天赋：arcane
科技天赋：tech
破甲天赋：armor_break
流血天赋：bleed
```

天赋不是单纯加数值，而是改变联动底色。

例如：

```text
雷纹天赋：
让部分升级卡触发电弧、跳电、连锁放电。

炎脉天赋：
让部分升级卡触发燃烧、爆燃、熔岩地面。

霜息天赋：
让部分升级卡触发减速、冰痕、冻结。

毒蚀天赋：
让部分升级卡触发毒雾、毒层扩散、腐蚀。
```

玩家看到的是：

```text
我带了雷纹天赋，所以我的分裂箭、枪芒、剑气都有机会出现雷电联动。
```

## 八、隐藏联动系统

联动不要做成卡。

代码提前写好联动表，玩家满足条件后自动触发。

### 代码结构示例

```ts
{
  id: "lightning_split_arrow",
  name: "雷羽分裂",
  requires: {
    upgrades: ["split_arrow"],
    talentTags: ["lightning"]
  },
  effect: "split_arrow_chain_lightning",
  description: "分裂小箭之间产生连锁电弧。"
}
```

玩家不需要看到“雷羽分裂”作为卡牌。
玩家只需要在战斗中看到效果，并在升级面板里看到提示。

## 九、第一批隐藏联动设计

### 1. 分裂箭 + 雷纹天赋 = 雷羽分裂

条件：

```text
拥有升级卡：分裂箭
拥有天赋标签：lightning
```

效果：

```text
分裂出来的小箭之间产生电弧；
小箭命中敌人时，会向附近目标跳电；
分裂箭等级越高，电弧数量或跳电范围越强。
```

视觉：

```text
蓝白电弧连接小箭或被命中的敌人；
命中时出现电火花。
```

### 2. 穿透箭 + 霜息天赋 = 霜痕穿透

条件：

```text
拥有升级卡：穿透箭
拥有天赋标签：ice
```

效果：

```text
箭矢穿透敌人后，在路径上留下冰痕；
擦到冰痕的敌人减速；
穿透等级越高，冰痕越长。
```

视觉：

```text
箭矢路径留下淡蓝冰线；
敌人脚下出现冰霜碎片。
```

### 3. 追踪箭 + 科技天赋 = 制导箭

条件：

```text
拥有升级卡：追踪箭
拥有天赋标签：tech
```

效果：

```text
追踪箭锁定角度更强；
优先攻击被标记敌人；
箭矢飞行时出现细小锁定线。
```

视觉：

```text
目标身上出现短暂锁定框；
箭矢带蓝白修正轨迹。
```

### 4. 爆裂箭 + 炎脉天赋 = 炎爆箭

条件：

```text
拥有升级卡：爆裂箭
拥有天赋标签：fire
```

效果：

```text
爆裂箭命中后留下短暂燃烧区域；
燃烧敌人死亡时可能产生小爆燃。
```

视觉：

```text
橙红爆炸圈；
地面短暂火焰残留。
```

### 5. 枪芒 + 雷纹天赋 = 雷贯枪芒

条件：

```text
拥有升级卡：枪芒
拥有天赋标签：lightning
```

效果：

```text
枪芒贯穿敌人后，在被贯穿敌人之间跳电；
枪芒等级越高，跳电越明显。
```

视觉：

```text
枪芒路径上有蓝白闪电；
被贯穿敌人之间出现电弧。
```

### 6. 枪芒 + 霜息天赋 = 寒星枪芒

条件：

```text
拥有升级卡：枪芒
拥有天赋标签：ice
```

效果：

```text
枪芒路径留下寒霜；
被枪芒多次命中的敌人可能被冻结。
```

视觉：

```text
枪芒尾迹带冰蓝晶尘。
```

### 7. 剑气 + 雷纹天赋 = 雷鸣剑气

条件：

```text
拥有升级卡：剑气
拥有天赋标签：lightning
```

效果：

```text
剑气命中后向附近敌人跳电；
剑气等级越高，跳电范围越强。
```

视觉：

```text
剑气边缘带电弧；
命中后出现蓝白闪光。
```

### 8. 剑气 + 毒蚀天赋 = 腐蚀剑气

条件：

```text
拥有升级卡：剑气
拥有天赋标签：poison
```

效果：

```text
剑气路径留下短暂毒雾；
毒雾中的敌人持续掉血。
```

视觉：

```text
绿色毒雾轨迹；
敌人身上有毒蚀粒子。
```

### 9. 地裂 + 炎脉天赋 = 熔岩地裂

条件：

```text
拥有升级卡：地裂
拥有天赋标签：fire
```

效果：

```text
地裂波变成熔岩裂缝；
裂缝短暂留在地上；
敌人踩上去持续受伤。
```

视觉：

```text
地面裂缝发红；
裂缝中有熔岩亮光。
```

### 10. 地裂 + 霜息天赋 = 冰裂大地

条件：

```text
拥有升级卡：地裂
拥有天赋标签：ice
```

效果：

```text
地裂命中敌人时减速；
地裂路径短暂结冰；
连续命中可能冻结。
```

视觉：

```text

```

### 11. 飞刃 + 毒蚀天赋 = 毒刃回旋

条件：

```text

拥有天赋标签：poison
```

效果：

```text
飞刃飞行路径留下毒雾；

```

视觉：

```text
回旋路径残留绿色毒雾。
```

### 12. 无人机协同 + 奥术天赋 = 奥术无人机

条件：

```text
拥有升级卡：无人机协同
拥有天赋标签：arcane
```

效果：

```text
无人机发射小型奥术弹；
奥术弹可触发魔法类命中特效。
```

视觉：

```text
无人机弹体由科技蓝变成紫蓝奥术弹。
```

## 十、流派阶段系统

流派阶段仍然要做，但不只看单一路线，还要看联动数量。

### 1. 纯武器流派阶段

统计：

```text
bowPoints
spearPoints
swordPoints
bladePoints
macePoints
magicPoints
techPoints
```

阶段建议：

```text
2 点：初成
4 点：成型
7 点：大成
```

示例：

```text
箭术初成
箭术成型
箭术大成

长枪初成
枪术成型
枪术大成

钝器发力
钝器成型
钝器霸体

奥术入门
奥术成型
奥术大成

科技起势
科技成型
科技过载
```

### 2. 天赋联动阶段

统计已激活的联动数量和联动类型。

例如：

```text
lightningSynergyCount
fireSynergyCount
iceSynergyCount
poisonSynergyCount
arcaneSynergyCount
techSynergyCount
```

当某种天赋联动达到一定数量，触发杂交大成。

示例：

```text
雷羽箭阵
熔岩地裂
雷贯枪阵
毒刃风暴
奥术无人机群
霜痕箭雨
```

## 十一、流派大成示例

### 1. 雷羽箭阵

条件：

```text
弓箭路线达到大成；
拥有分裂箭；
拥有雷纹天赋；
已激活雷羽分裂。
```

效果：

```text
箭雨也会附带电弧；
分裂小箭之间连锁放电；
追踪箭优先攻击带电敌人。
```

视觉：

```text
蓝白雷羽箭阵；
屏幕局部出现雷电箭雨。
```

### 2. 熔岩地裂

条件：

```text
钝器路线达到成型或大成；
拥有地裂；
拥有炎脉天赋；
已激活熔岩地裂。
```

效果：

```text
地裂范围扩大；
熔岩裂缝停留更久；
重击有概率触发连环地裂。
```

视觉：

```text
地面多段红色裂缝；
敌人脚下喷出火光。
```

### 3. 雷贯枪阵

条件：

```text
长枪路线达到成型或大成；
拥有枪芒；
拥有雷纹天赋；
已激活雷贯枪芒。
```

效果：

```text
枪芒贯穿路径更长；
枪芒命中的敌人之间跳电；
周期性生成额外枪影。
```

视觉：

```text
枪芒带雷电残影；
贯穿敌人后有蓝白电链。
```

### 4. 毒刃风暴

条件：

```text
飞刃路线达到成型或大成；
拥有飞刃回旋相关升级；
拥有毒蚀天赋；
已激活毒刃回旋。
```

效果：

```text
飞刃路径留下毒雾；
毒雾范围扩大；
飞刃返回时再次触发毒爆。
```

视觉：

```text
绿色回旋轨迹；
毒雾环绕玩家和敌人。
```

## 十二、命中反馈系统

战斗不爽还有一个原因是命中反馈弱。

每次命中至少应有一种反馈：

```text
命中火花
受击闪白
轻微击退
伤害数字
暴击特殊颜色
特殊攻击命中特效
```

### 不同攻击命中特效

```text
箭矢：
木屑 / 刺入火花。

爆裂箭：
小爆炸冲击环。

魔法弹：
紫蓝粒子爆开。
```

## 十七、最终验收标准

完成整个计划后，玩家应该能明显感受到：

```text
1. 每种武器打出去都不一样。
2. 近战武器前期是真的近战。
3. 拿到枪芒、剑气、地裂后，近战才自然获得远程派生能力。
4. 升级卡重复获得会进入 I / II / III 阶段。
5. 元素不是卡池垃圾，而是来自天赋。
6. 升级卡和天赋会自动产生隐藏联动。
7. 玩家不会被复杂卡种弄晕，但战斗会自然出现组合效果。
8. 中后期能感受到流派初成、成型、大成。
9. 命中反馈明显增强，攻击更爽。
10. 最终构筑有名字、有视觉、有联动、有毕业感。
```

核心方向：

```text
升级卡负责武器机制；
天赋负责元素倾向；
联动由代码自动触发；
流派阶段负责成型反馈；
玩家只看到自然产生的爽感。
```
