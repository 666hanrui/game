## 一、核心定位

一款**俯视 2D PVE Roguelite 射击游戏**，灵感来自《弓箭手大作战》的升级随机三选一机制，在此基础上增加种族和体系两层构筑。

- **不是**：IO 对战、回合制地牢、Vampire Survivors 换皮
- **目标平台**：手机端（双摇杆触控）+ PC（键盘鼠标）

---

## 二、三层构筑体系

### 种族（开局选择）

| 种族 | 特性 | 颜色 | 血量 | 速度 | 伤害 | 经验 | 特殊 |
|------|------|------|------|------|------|------|------|
| 人类 | 平衡全能 | 蓝 #4fc3f7 | 1.0 | 1.0 | 1.0 | +20% | 升级更快 |
| 精灵 | 高敏低血 | 绿 #81c784 | 0.75 | 1.25 | 0.9 | 1.0 | 半径 -3 |
| 兽人 | 血厚攻高 | 橙 #ff7043 | 1.4 | 0.85 | 1.15 | -10% | 受伤后伤害 +30% |

### 体系（首次升级锁定，不可更改）

| 体系 | 定位 | 颜色 | 主题 |
|------|------|------|------|
| ⚙️ 科技系 | 远程火力 · 召唤 · AOE | 蓝 #42a5f5 | 无人机、追踪导弹、电磁脉冲 |
| 👊 古武系 | 近战强化 · 暴击 · 生存 | 红 #ef5350 | 铁布衫、寸拳、连斩 |
| 🔮 魔法系 | 元素效果 · 控制 · 爆发 | 紫 #ab47bc | 冰霜、火球、闪现 |

### 技能（每次升级三选一）

**中立（4 个）**：生命强化、快速射击、强力箭、疾跑

**科技（8 个）**：双发芯片、无人机、电磁脉冲、弹夹扩容、追踪模组、超载、破片弹、激光瞄准

**古武（8 个）**：铁布衫、寸拳、心眼、连斩、金刚体、嗜血、重击、肾上腺素

**魔法（8 个）**：冰霜箭、火球术、法力盾、电磁场、闪现、连锁闪电、奥术强化、冰霜护甲

每个技能有稀有度（普通/稀有/史诗）、可叠加次数上限、属性修正和特殊效果标签。

## 三、核心游戏循环

```
开局 → 种族三选一
  ↓
进入 2400×2400 俯视世界（Camera 跟随玩家）
  ↓
地图每 3 秒随机刷新经验宝石
WASD/左摇杆移动 · 鼠标/右摇杆瞄准射击
  ↓
敌人从四边涌入：🟢史莱姆(歪椭圆) 🟤蜘蛛(六边形) ⚪骷髅(菱形)
  ↓
击杀敌人 → 经验条攒满 →
  ├─ 首次升级 → 体系三选一（锁定）
  └─ 后续升级 → 技能卡三选一（55%体系卡 + 25%中立 + 20%其他）
      ↓
  属性修饰器链叠加（攻击/攻速/血量/暴击/弹射物数量）
      ↓
  波次清空 → 下一波（敌人数/血量/速度递增）
      ↓
  死亡 → 结算画面
```

## 四、视觉风格

清晰几何体 + 颜色区分，不使用外部素材：

- **玩家**：种族色圆形 + 背后弧形弓 + 白色三角方向标
- **敌人**：歪椭圆（史莱姆）、正六边形（蜘蛛）、菱形（骷髅）
- **箭矢**：旋转矩形箭身 + 三角形箭头
- **掉落物**：菱形钻石（蓝=经验 / 绿=血包）
- **世界**：深色网格地面 + 可见边界框

## 五、技术栈与架构

| 层 | 选型 |
|----|------|
| 语言 | TypeScript |
| 构建 | Vite 6 |
| 渲染 | Canvas 2D |
| 部署 | 静态 HTML，零后端 |

```
src/
├── main.ts              # 入口
├── core/
│   ├── Game.ts          # 主循环、碰撞、流程控制
│   ├── Camera.ts        # 世界坐标→屏幕坐标
│   └── Input.ts         # 键盘+鼠标 / 双摇杆触控
├── entities/
│   ├── Player.ts        # 玩家
│   ├── Enemy.ts         # 敌人（三类型）
│   ├── Projectile.ts    # 弹射物
│   └── Pickup.ts        # 掉落物
├── systems/
│   ├── WaveSystem.ts    # 波次管理
│   ├── CombatSystem.ts  # 伤害 + 碰撞
│   └── XPLevelSystem.ts # 经验 + 升级
├── data/
│   ├── races.ts         # 种族数据
│   ├── schools.ts       # 体系数据
│   └── skills.ts        # 28 技能池
├── ui/
│   ├── HUD.ts           # 血条/经验/波次/种族/体系
│   └── UpgradePanel.ts  # 种族面板/体系面板/技能卡面板
└── utils/
    └── math.ts          # Vec2 向量/碰撞/随机/对象池
```
